## 💡 Pro Tips

1. **Consistency > Intensity**: Don't skip days
2. **Quality > Quantity**: Understand patterns, don't just solve
3. **Active Recall**: Revise Saturdays religiously
4. **Time Yourself**: Practice with 45-min timer from Week 5
5. **Mock Interviews**: Start from Week 8
6. **Join Communities**: Discuss on Discord, Telegram, Reddit
7. **Track Progress**: Use Excel/Notion to mark completed
8. **Adjust if Needed**: If falling behind, reduce to 2 problems/day
9. **Theory First on Weekends**: Get theory done early in the day
10. **Stay Healthy**: Sleep 7-8 hours, exercise, eat well
